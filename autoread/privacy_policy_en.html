2. Information We Collect

Important: AutoRead does NOT collect, store, or transmit any personal information to external servers. All data processing occurs locally on your device.

2.1 Usage Data

  • AutoRead needs to record automation operation configurations you set
  • All configuration data is stored entirely on your device
  • Automation scripts and scheduled tasks run only locally
  • We do NOT upload any usage data to external servers

2.2 Local Storage

  • The app stores automation configurations, gesture paths and other data in the device's local database
  • You can clear these configurations at any time through the app
  • History records are saved locally and will be deleted when the app is uninstalled

3. Permissions We Request

3.1 Accessibility Service Permission

Important Disclosure: Accessibility Service Usage

Why We Need This Permission:

  • Enable automatic screen swiping and gesture operations
  • Allow hands-free content browsing
  • This is the foundation of the app's core functionality

Specific Features Using This Permission:

  • Automatic screen swiping (up, down, left, right directions)
  • Custom gesture path execution
  • Scheduled automation tasks
  • Repeat action automation

How We Use Accessibility Service:

  • Only executes actions YOU explicitly configure: The app will not automatically perform any actions you haven't set
  • Only active when YOU start tasks: The service only runs when you actively start automation tasks
  • Does NOT collect any data: Does not read, collect, or transmit your screen content, personal information, or app data
  • Local processing: All operations are completed on your device locally, no network connection needed
  • No background activity: Accessibility service does not remain active in background after app is closed or tasks are stopped

Your Control:

  • You can disable this service anytime through System Settings → Accessibility → AutoRead
  • The app will not be able to perform automation functions after disabling
  • You have complete control over when to start and stop automation tasks

Privacy Commitment:

  • We will NOT use accessibility service to collect your personal information
  • We will NOT read or record your screen content
  • We will NOT track your app usage behavior
  • We will NOT access or transmit data from other apps
  • We will NOT perform any operations without your permission

Technical Details:

  • Usage Scenario: Only performs swipe operations within apps you specify
  • Permission Scope: Limited to executing touch and swipe gestures only
  • Data Access: Does not access any app content or user data

3.2 Overlay Permission

  • Purpose: Display floating ball controller for quick start/stop of automation tasks
  • Use Case: Show floating control buttons on top of other apps

3.3 Internet Permission

  • Purpose: Display this Privacy Policy and Terms of Service
  • Use Case: Only used when you choose to view legal documents within the app
  • Note: Does not transmit any user data or usage records via network

3.4 Other Permissions

  • Keep Screen On: Keep screen awake during long-running automation tasks
  • Vibrate: Provide haptic feedback when tasks complete
  • Set Alarm: Used for scheduled task functionality
